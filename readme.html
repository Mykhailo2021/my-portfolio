ReadMe - My Portfolio

Deliverables

A zip file should be uploaded to Moodle, which contains the entire contents of your website. You also need to have a separate web page that outlines the following:

  1. List all the Bootstrap and Project features you added
  2. The URL of where your website is hosted

Bootstrap Features Used

  • Responsive Navbar with a toggler for mobile views
  • Carousel component for showcasing featured projects
  • Cards for displaying project details and educational background
  • Buttons with hover effects for interactivity
  • Grid system for responsive layout across devices
  • Forms for the contact section with validation
  • Modals for displaying additional information about projects and certificates
  • Scrollspy for highlighting active navigation links
  • Utility classes for spacing, text alignment, and visibility

JavaScript Features Integrated

  • Dynamic text changing in the hero section using JavaScript for a more engaging user experience.
  • Scroll event listener to reveal the "About Me" section when the user scrolls down the page.
  • Project page navigation with smooth transitions between different project pages using JavaScript.
  • Auto-scroll functionality for project pages, allowing users to view projects automatically after a set interval.
  • Modal functionality for displaying project details and certificates, enhancing user interaction.
  • Google Maps integration for displaying locations related to education and work experience.
  • Music player modal with play, pause, next, and previous track functionality, enhancing the user experience.
  • Dynamic loading of certificate details in a modal when a certificate image is clicked, providing a seamless experience.
